## Installation

1. Install Node
1. Download or clone repository
1. Open a terminal in the downloaded folder
1. Install dependencies `npm install`
1. Create `.env` file. Set `MONGODB_URI` and `JWT_SECRET`

## Usage

1. Run using this command: `npm start`
